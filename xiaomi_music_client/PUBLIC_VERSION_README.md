# 小爱音乐客户端 - 公开版本说明

## 版本信息
- **分支**: `release/1.0.2-public`
- **基于**: `release/1.0.2`
- **类型**: 公开发布版本

## 与私人版本的主要差异

### ❌ 移除的功能

1. **内置JS音源**
   - 移除了预置的"小秋音乐"脚本
   - 移除了硬编码的fallback音源URL
   - 用户需要自行添加JS音源脚本

2. **YouTube代理功能**
   - 完全移除YouTube代理搜索功能
   - 删除相关的下载源配置
   - 移除YouTube音频质量设置

### ✅ 保留的功能

1. **统一API音源**
   - 默认使用 `https://music.txqq.pro` 统一API
   - 支持QQ音乐、网易云音乐等主流平台
   - 稳定可靠的音乐搜索和播放

2. **JS音源支持**
   - 支持用户自行导入JS脚本
   - 支持从URL或本地文件加载脚本
   - 完整的JS执行环境

3. **完整的播放功能**
   - 音乐搜索、播放、歌单管理
   - TTS文字转语音
   - 设备控制和同步

## 如何使用

### 1. 基础使用（推荐）
默认配置下，应用会使用统一API进行音乐搜索和播放，无需额外配置。

### 2. 添加JS音源（可选）
如果需要使用JS音源，请：

1. 进入 **设置** → **音源设置**
2. 选择 **JS脚本管理**
3. 通过以下方式添加脚本：
   - **从URL导入**：输入有效的JS脚本URL
   - **从本地文件导入**：选择本地的.js文件

### 3. 音源优先级
应用会按以下优先级使用音源：
1. 统一API（默认）
2. 用户添加的JS脚本（如果启用）

## 注意事项

- 本版本不包含任何预置的音源脚本
- YouTube相关功能已完全移除
- 如需使用JS音源，请确保脚本来源的合法性和安全性
- 建议优先使用统一API，稳定性更好
